Emitter defines

The function defines that require a buffer to return any useful information will return false if there is no buffer in the emitters buffer stack. 
aaxEmitterIsValid(handle)
aaxIsValid(handle, AAX_EMITTER)
Return true if the handle is a valid emitter handle, false otherwise. 
aaxEmitterGetSetup(emitter, type)
aaxBufferGetSetup(aaxEmitterGetBuffer(emitter), type)
Return the value of the requested setup type.
See aaxBufferGetSetup for more information. 
aaxEmitterGetBuffer(emitter)
aaxEmitterGetBufferByPos(emitter, 0, false)
Return the first buffer of the emitters buffer stack, the buffer will not be removed from the stack.

Since the buffer will not be copied it may become invalid when the last reference to it has been destroyed.

aaxEmitterGetNoTracks(emitter)
aaxEmitterGetSetup(emitter, AAX_TRACKS)
Return the number of tracks in the buffer associated with this emitter. 
aaxEmitterGetFormat(emitter)
aaxEmitterGetSetup(emitter, AAX_FORMAT)
Return the format of the buffer associated with this emitter. 
aaxEmitterGetGain(emitter)
Return the linear gain value of this emitter using the volume-filter. 
aaxEmitterGetGainMinMax(emitter, min_gain, max_gain)
Return the linear minimum and maximum gain allowed by the emitter using the volume-filter and store it in min_gain and max_gain.
If either min_gain or max_gain is NULL it's value will not be set. 
aaxEmitterGetPitch(emitter)
Return the pitch factor of this emitter using the pitch-effect. 
aaxEmitterGetLooping(emitter)
aaxEmitterGetMode(emitter, AAX_LOOPING)
Return whether this emitter is set to looping or not. 
aaxEmitterGetOffsetBytes(emitter)
aaxEmitterGetOffset(emitter, AAX_BYTES)
Return the playback offset in bytes from the start of the first buffer in the buffer queue. 
aaxEmitterGetOffsetSamples(emitter)
aaxEmitterGetOffset(emitter, AAX_SAMPLES)
Return the playback offset in samples from the start of the first buffer in the buffer queue. 
aaxEmitterGetOffsetFrames(emitter)
aaxEmitterGetOffset(emitter, AAX_FRAMES)
Return the playback offset in frames from the start of the first buffer in the buffer queue.
