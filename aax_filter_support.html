int aaxFilterGetSlot(aaxFilte filter, unsigned slot, int ptype,
                                              float *p1, float *p2, float *p3)
Return all parameters of a defined parameter-slot for a filter.

  • The filter parameter is the handle as returned by aaxFilterCreate or one of the class getter functions.
  • The slot parameter defines which slot number to return.
    Slot numbers start at zero.
  • The ptype parameter specifies the unit of the parameter and is type specific.
    See enum aaxType for more information.

Return the value if successful or AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed filter handle or an incorrect slot number. 

float aaxFilterGetParam(aaxFilter filter, int param_no, int ptype)
Get the value of the specified parameter of the filter.

  • The filter parameter is the handle as returned by aaxFilterCreate or one of the class getter functions.
  • The param_no parameter defines which value of the filter parameters to return.
    See enum aaxParameter for more information.
  • The ptype parameter specifies the unit of the parameter and is type specific.
    See enum aaxType for more information.

Return the value if successful or AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed filter handle or an incorrect param_no value.
