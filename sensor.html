Class Sensor

Header file: <aax/aeonwave.hpp>
Namespace: aax

 

The Sensor class encapsulates Driver, Mixer and Sensor functions.

The sensor represents the user in the 3D virtual world and is a virtual recording device which has a position, orientation and a velocity. The position of the sensor relative to all emitters influences the loudness of every individual sound emitter. The orientation of the sensor in relation to every individual sound emitter influences where that emitter is placed in the users stereo perception. The velocity of the sensor affects the Doppler effect.
