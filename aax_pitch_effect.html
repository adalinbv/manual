Pitch Effect

Adjusting the pitch of an audio-frame or mixer object with multiple registered emitters using the pitch effect will affect the pitch of all registered emitters. The effect makes it possible to let the library clamp the pitch to a maximum value.

The hard maximum pitch-limit is 2.0 and cannot be altered.

The Pitch effect is supported by the Mixer, Emitters and Audio-Frames.
Pitch Effect parameters
The effect uses only one effect slot.
AAX_PITCH_EFFECT
STEREO
p0
AAX_PITCH
Current pitch multiplier.
Between 0.0 and 2.0, defaults to 1.0
p1
AAX_MAX_PITCH
Maximum pitch value, anything higher will be saturated.
Between 0.0 and 2.0, defaults to 2.0
p2
unused
p3
unused
Pitch Effect state options
The effect is always enabled.
