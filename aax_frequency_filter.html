Frequency Filter

Applying a Frequency Filter to an audio-frame or mixer object with multiple registered emitters will affect the frequency response of all registered emitters. This makes it possible to simulate obstruction or occlusion of a group of emitters with one applied filter.

Applying the frequency filter to the Scenery can be used for obstruction or occlusion of all emitters in the scene (in other words to make the player seek shelter in or under an obstacle).

The Envelope Following mode can be used for Auto-Wah and can also be used in the Inverted mode.

The Timed Transition mode is a one time operation which moves from a start value to the end value in the time specified by the sweep rate (1/rate seconds). Once the end value is reached it will stay at that value.

If the value of the HF Cutoff Frequency parameter is below the value of the Cutoff Frequency parameter then both settings will be swapped.

Filter type selection and filter order selection are available for AeonWave 2.6 (filter version 1.1 ).

AeonWave 3.7.2 (version 1.11 of the filter) adds support for a timed-transition of the filter frequency.

AeonWave 3.8.0 (version 1.12 of the effect) adds support for logarithmic sliding.

AeonWave 3.8.2 (version 1.13 of the effect) adds support for randomness.

The Frequency filter is supported by the Scenery, Emitters and Audio-Frames.

Frequency Filter state options
Specify a filter type and/or a filter order or a waveform type to enable the filter or false to disable it.

The fastest filters are 6dB/Oct and 12dB/Oct (both are equally fast). Higher order filters require extra filter passes. This makes an 8th order filter 4 times slower than a 2nd order filter.

Filter type selection and filter order selection are available for AAX_frequency_filter version 1.1 or higher only.
The filter type and filter order can be combined by using the bitwise OR operation.
The default for true is (AAX_2ND_ORDER|AAX_BUTTERWORTH)
