AAX_POSITION
The aaxEmitterMode type mode parameter defines how the source is located in relation to the sensor:
  • AAX_STEREO: No positional information will be used.
  • AAX_ABSOLUTE: The position, orientation and velocity are absolute.
  • AAX_RELATIVE: The position, orientation and velocity are relative to the sensors position, orientation and velocity.
AAX_LOOPING
The mode parameter defines whether to enable looping or not.
  • false disables looping.
  • true enables looping with an infinite number of loops.
  • mode > 1 enables looping with 'mode' number of loops before continuing.
AAX_BUFFER_TRACK
The mode parameter defines the track-number to use from a multi-track buffer when in 3d mode. Setting the buffer track to a number greater than the number of tracks in the buffer will result in an error.
