Effect initialization

bool aaxEffectSetSlot(aaxEffect effect, unsigned slot, int ptype,
                                              float p1, float p2, float p3, float p4)
Same as aaxEffectSetSlotParams except that the fields of the values parameter have to be specified independently.

  • p1 defines values[0], p2 defines values[1], p3 defines values[2] and p4 defines values[3]. 
bool aaxEffectSetSlotParams(aaxEffect effect, unsigned slot, int ptype, aaxVec4f values)
Set the values of a parameter-slot for an effect.
Most effects need just one parameter-slot but some require more slots to be set properly.

  • The effect parameter is the handle as returned by aaxEffectCreate or one of the class getter functions.
  • The slot parameter defines which slot number to alter.
    Slot numbers start at zero.
  • The ptype parameter specifies the unit of the values parameter and is type specific.
    See enum aaxType for more information.
  • The values parameter is an array of four float type parameters that represents the effect-type specific effect parameters.
    See aaxEffectType for more information.
  • If one or more of the fields of the values parameter is NaN it's value will not bet set.
Returns true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed effect handle or an incorrect slot number. 

bool aaxEffectSetParam(aaxEffect effect, int param_no, int ptype, float value)
Set the value of the specified parameter of the effect.

  • The effect parameter is the handle as returned by aaxEffectCreate or one of the class getter functions.
  • The param_no parameter defines which one of the parameters to alter.
    See enum aaxParameter for more information.
  • The ptype parameter specifies the unit of the parameter and is type specific.
    See enum aaxType for more information.
  • If the value parameter is NaN the parameter value will not bet set.

Returns true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed effect handle, an incorrect param_no value or the value parameter is a NaN. 

bool aaxEffectSetState(aaxEffect effect, uint64_t state)
Change the state of the effect.

  • The effect parameter is the handle as returned by aaxEffectCreate or one of the class getter functions.
  • Possible values for the state parameter differ between effect types
    See aaxEffectState for more information.

Returns true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed effect handle. 

bool aaxEffectAddBuffer(aaxEffect effect, aaxBuffer buffer)
Adds an audio buffer to the effect.

  • Adding an audio buffer to an effect is only useful for the convolution effect.
  • Only buffers with a single track are supported.
Return true if successful, false otherwise.

Reasons for failure are an incorrect or already destroyed effect or buffer handle, adding a buffer which has no data assigned to it or adding a buffer with more than one tracks.
