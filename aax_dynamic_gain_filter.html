Dynamic Gain Filter

The Dynamic Gain filter constantly adjusts the objects volume without any interaction. Adjusting can be based on time functions (tremolo) using several waveform types or based on the gain of the input signal (envelope following) making the audio sound snappier.

The filter combines the Dynamic Gain filter of the current sound source and of it's mixer parent. This makes it possible to continually decrease and increase the filters strength.

If the combined Offset parameter and Depth parameter exceeds 1.0 then the depth of the low frequency oscillator is automatically clamped.

Version 3.6.0 of AeonWave (version 1.01 of the effect) adds support for an initial delay before the effect kicks in.

AeonWave 3.8.2 (version 1.02 of the effect) adds support for randomness.

The Dynamic Gain filter is supported by Emitters, Audio-Frames and the Mixer.
Dynamic Gain Filter parameters
The filter uses only one filter slot.
AAX_DYNAMIC_GAIN_FILTER
STEREO
p0
AAX_INITIAL_DELAY
Delay before the effect kicks in.
Between 0.0 and 10.0 seconds.
p1
AAX_LFO_FREQUENCY
Frequency of the low-frequency oscillator for tremolo.
Response rate in Hertz (1/seconds) when using Envelope Following mode.
Between 0.01 and 50 Hz.
p2
AAX_LFO_DEPTH
Depth range of the low-frequency oscillator.
Between 0.0 and 1.0.
p3
AAX_LFO_OFFSET
Minimum offset value of the low-frequency oscillator.
Between 0.0 and 1.0
Dynamic Gain Filter state options
Specify a waveform type or envelope-following to enable the dynamic filter or false to disable it.
AAX_DYNAMIC_GAIN_FILTER
Valid waveform types are:
AAX_CONSTANT_VALUE (or true)
AAX_TRIANGLE_WAVE
AAX_SINE_WAVE
AAX_SQUARE_WAVE
AAX_SAWTOOTH_WAVE
AAX_CYCLOID_WAVE
AAX_ENVELOPE_FOLLOW
AAX_INVERSE_TRIANGLE_WAVE
AAX_INVERSE_SINE_WAVE
AAX_INVERSE_SQUARE_WAVE
AAX_INVERSE_SAWTOOTH_WAVE
AAX_INVERSE_CYCLOID_WAVE
AAX_INVERSE_ENVELOPE_FOLLOW

All waveforms may be combined with AAX_ENVELOPE_FOLLOW_LOG or AAX_INVERSE_ENVELOPE_FOLLOW_LOG to create exponential behaviour, e.g. AAX_ENVELOPE_FOLLOW|AAX_ENVELOPE_FOLLOW_LOG
