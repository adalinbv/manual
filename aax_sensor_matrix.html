Position and Orientation

int aaxSensorSetMatrix64(aaxConfig config, const aaxMtx4d mtx)
Set the position-and-orientation matrix.

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The sensor matrix should be inversed using aaxMatrix64Inverse first if the matrix was manipulated using the matrix support functions.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle, a NULL pointer for the mtx parameter or a matrix that contains one ore more NaN's. 


int aaxSensorGetMatrix64(const aaxConfig config, aaxMtx4d mtx)
Stores the position-and-orientation matrix int mtx.

  • The config parameter is the handle as returned by aaxOpenDevice.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle or a NULL pointer for the mtx parameter.
