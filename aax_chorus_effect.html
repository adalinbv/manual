Chorus Effect

The Chorus effect adds a delayed copy of the original signal to the undelayed signal.
This gives a fuller sound an is perceived as two versions of the same sound being played simultaneously.

Only one delay effect can be used at an object (mixer, emitter, audio-frame) at any given time: either Phasing, Chorus or Flanging.

In addition to following a continuous Low Frequency Oscillator level for effect strength it is also possible to alter the strength of the effect by following the level of the input signal. This makes it possible to have a strong effect for high input signal levels and no effect for low input levels or vice versa.

If the combined Offset parameter and Depth parameter exceeds 1.0 then the depth of the low frequency oscillator is automatically clamped.

Starting with AeonWave version 3.6 the code supports feedback and a frequency filter to be applied to the delayed signal before mixing it with the original. Feedback is used for adding flanging to the signal before mixing with the delayed signal.

The Chorus effect is supported by the Mixer, Emitters and Audio-Frames.
Chorus Effect parameters
The effect uses only one effect slot.
AAX_CHORUS_EFFECT
STEREO
Slot 0
p0
AAX_DELAY_GAIN
Gain factor for the delayed signal.
AAX_LINEAR, AAX_LOGARITHMIC

Between 0.0 and 1.0
p1
AAX_LFO_FREQUENCY
Frequency of the low-frequency oscillator.
Between 0.01 and 10 Hz.
p2
AAX_LFO_DEPTH
Depth of the low-frequency oscillator.
Between 0.0 and 1.0, set to 0.0 for a constant offset.
p3
AAX_LFO_OFFSET
Offset position of the low-frequency oscillator.
Between 0.0 and 1.0
Slot 1
p0
AAX_DELAY_CUTOFF_FREQUENCY
Cut-off frequency for the delayed signal in Hz.
Between 20 Hz and 22050 Hz
p1
AAX_DELAY_CUTOFF_FREQUENCY_HF
Higher cut-off frequency for the delayed signal in Hz.
Between 20 Hz and 22050 Hz
p2
AAX_FEEDBACK_GAIN
Gain of the feedback signal.
Between 0.0 and 1.0
p3
AAX_DELAY_RESONANCE
Resonance factor of the frequency filter.
Between 0.1 en 100.0, defaults to 1.0
Chorus Effect state options
Specify a waveform type or envelope-following to enable the chorus effect or AAX_FALSE to disable it.
AAX_CHORUS_EFFECT
Valid waveform types are:
AAX_CONSTANT_VALUE (or AAX_TRUE)
AAX_TRIANGLE_WAVE
AAX_SINE_WAVE
AAX_SQUARE_WAVE
AAX_SAWTOOTH_WAVE
AAX_ENVELOPE_FOLLOW
AAX_INVERSE_TRIANGLE_WAVE
AAX_INVERSE_SINE_WAVE
AAX_INVERSE_SQUARE_WAVE
AAX_INVERSE_SAWTOOTH_WAVE
AAX_INVERSE_ENVELOPE_FOLLOW
