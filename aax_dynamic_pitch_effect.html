Dynamic Pitch Effect

The Dynamic Pitch effect constantly adjusts the objects pitch without any interaction. Adjusting is based on time functions (vibrato) using several waveform types.

The effect combines the Dynamic Pitch effect of the current sound source and of it's mixer parent. This makes it possible to continually decrease and increase the effects strength.

If the combined Offset parameter and Depth parameter exceeds 1.0 then the depth of the low frequency oscillator is automatically clamped.

Version 3.6.0 of AeonWave (version 1.01 of the effect) adds support for an initial delay before the effect kicks in.

AeonWave 3.8.2 (version 1.02 of the effect) adds support for randomness.

The Dynamic Pitch effect is supported by the Mixer, Emitters and Audio-Frames.
Dynamic Pitch Effect parameters
The effect uses only one effect slot.
AAX_DYNAMIC_PITCH_EFFECT
STEREO
p0
AAX_INITIAL_DELAY
Delay before the effect kicks in.
Between 0.0 and 10.0 seconds.
p1
AAX_LFO_FREQUENCY
Frequency of the low-frequency oscillator.
Between 0.01 and 50 Hz.
p2
AAX_LFO_DEPTH
Depth of the low-frequency oscillator.
Between 0.0 and 1.0
p3
AAX_LFO_OFFSET
Offset position of the low-frequency oscillator.
Between 0.0 and 1.0
Dynamic Pitch Effect state options
Specify a waveform type to enable the dynamic effect or AAX_FALSE to disable it.
AAX_DYNAMIC_PITCH_EFFECT
Valid waveform types are:
AAX_CONSTANT_VALUE (or AAX_TRUE)
AAX_TRIANGLE_WAVE
AAX_SINE_WAVE
AAX_SQUARE_WAVE
AAX_SAWTOOTH_WAVE
AAX_CYCLOID_WAVE
AAX_ENVELOPE_FOLLOW
AAX_INVERSE_TRIANGLE_WAVE
AAX_INVERSE_SINE_WAVE
AAX_INVERSE_SQUARE_WAVE
AAX_INVERSE_SAWTOOTH_WAVE
AAX_INVERSE_CYCLOID_WAVE
AAX_INVERSE_ENVELOPE_FOLLOW

All waveforms may be combined with AAX_ENVELOPE_FOLLOW_LOG or AAX_INVERSE_ENVELOPE_FOLLOW_LOG to create exponential behaviour, e.g. AAX_ENVELOPE_FOLLOW|AAX_ENVELOPE_FOLLOW_LOG
