Mixer defines

aaxMixerGetGain(config)
Return the linear gain value of the mixer using the volume-filter. 
aaxMixerGetPich(config)
Return the pitch factor of the mixer using the pitch-effect. 
aaxMixerGetFrequency(config)
aaxMixerGetSetup(config, AAX_FREQUENCY)
Return the sample frequency of the mixer. 
aaxMixerGetRefreshRate(config)
aaxMixerGetSetup(config, AAX_REFRESHRATE)
Return the refresh rate of the mixer. 
aaxMixerGetNoMonoSources()

Return the number of mono emitters that are supported by this driver configuration.
