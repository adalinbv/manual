Driver initialization

aaxConfig aaxDriverOpenDefault(enum aaxRenderMode mode)
Open the default driver in the requested mode.
  • the mode parameter is an enumerated type aaxRenderMode.
Return a driver configuration handle. 
aaxConfig aaxDriverOpen(aaxConfig config)
Open the driver using the provided driver configuration in the mode specified by the handle.

Return a driver configuration handle. 

aaxConfig aaxDriverOpenByName(const char* name, enum aaxRenderMode mode)
Open the driver that is specified by the name parameter in the requested mode.
  • the name parameter consists of the backend name, possibly followed by " on " and a output port name.
    if name is NULL the default device will be used.
  • the mode parameter is an enumerated type aaxRenderMode.
Return a driver configuration handle. 
bool aaxDriverClose(aaxConfig config)
Close the driver configuration.

Return true if successful, false otherwise.
Reasons for failure are an invalid driver handle or trying to close a driver that is still registered. 

bool aaxDriverDestroy(aaxConfig config)
Free all the resources in use by the driver configuration.

Driver configurations that are opened using one of the aaxDriverOpen functions should be closed using aaxDriverClose first.

Return true if successful, false otherwise.
Reasons for failure are an invalid driver handle or trying to close a driver that is still registered. 

bool aaxDriverSetSetup(const aaxConfig config, enum aaxSetupType type, const char *setup)

These functions change the respective setup.

  • The config parameter is the handle as returned by aaxDriverOpen.
  • The type parameter is an enumerated type and specifies which setup parameter to change.
  • The setup parameter is type parameter specific (see aaxSetupType for more information).

Return true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed class handle or an incompatible setup value.

 

If hardware limitations prevent a particular request then a closest match will be selected.
