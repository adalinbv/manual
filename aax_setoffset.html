Buffer Offset

unsigned long aaxSensorGetOffset(const aaxConfig config, enum aaxType type)
unsigned long aaxEmitterGetOffset(const aaxEmitter emitter, enum aaxType type)
  • The config parameter is the handle as returned by aaxOpenDevice.
  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • The type parameter is an enumerated type aaxType and describes the unit-type for the buffer offset.

Return the offset from the start of the first buffer in the buffer stack or false in case of an error.
