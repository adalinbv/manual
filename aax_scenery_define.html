Scenery configuration

aaxScenerySetSoundVelocity(config, speed)
Set the sound velocity of the scenery using the velocity-effect.
Should be in the same units as the velocity vectors. 
aaxScenerySetDopplerFactor(config, doppler)
Set the Doppler factor of the scenery using the velocity-effect.
Setting it to something other than 1.0 will exaggerate or understate the Doppler effect. 
aaxScenerySetDistanceModel(config, model)
Set the distance model type for the scenery using the distance-effect. 
aaxSceneryEnableFrequencyFilter(config, enable)
Enable or disable the frequency filter for the scenery. 
aaxScenerySetFrequencyFilter(config, cutoff, lf_gain, hf_gain)
Set the frequency filters cut-off frequency and linear low-frequency and high-frequency gain for the scenery. 


aaxSceneryGetSoundVelocity(config)
Return the sound velocity setting of the scenery using the velocity-effect.
aaxSceneryGetDopplerFactor(config)
Return the Doppler factor of the scenery using the velocity-effect.
