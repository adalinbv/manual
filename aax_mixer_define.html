Mixer defines

Emitters and audio-frames that are not registered will not be affected by any of these defined functions. 
aaxMixerInit(config)
aaxMixerSetState(config, AAX_INITIALIZED)
Initialize the mixer using the current setup parameters.
It is possible to alter the mixer setup prior to initializing the mixer. 
aaxMixerStart(config)
aaxMixerSetState(config, AAX_STATUS_PLAYING)
Start playback of the registered emitters and audio frames. 
aaxMixerSuspend(config)
aaxMixerSetState(config, AAX_STATUS_SUSPENDED)
Temporarily suspend playback of the registered emitters and audio frames.
The mixer configuration will not be altered. 
aaxMixerResume(config)
aaxMixerSetState(config, AAX_STATUS_PLAYING)
Restart playback of the suspended and registered emitters and audio frames. 
aaxMixerStop(config)
aaxMixerSetState(config, AAX_STATUS_STOPPED)
Permanently stop playback of the registered emitters and audio frames.
The mixer configuration will be set to initialized but the setup will not be affected. 
aaxMixerSetGain(config, gain)
Set the linear gain of the mixer using the volume-filter.
This will act as a master gain. 
aaxMixerSetPitch(config, pitch)
Set the pitch factor of the mixer using the pitch-effect.
This will act as a global pitch shifter. 
aaxMixerSetNoTracks(config, tracks)
aaxMixerSetSetup(config, AAX_TRACKS, tracks)
Set the number of output tracks for this configuration.

aaxMixerSetFrequency(config, frequency)
aaxMixerSetSetup(config, AAX_FREQUENCY, frequency)
Set the mixing frequency for this configuration.

aaxMixerSetRefreshRate(config, refrate)
aaxMixerSetSetup(config, AAX_REFRESHRATE, refrate)
Set the frequency at witch the internal parameters, like position changes, get processed.
A higher refresh rate means shorter response times at the cost of a higher processor load. 
aaxMixerSetFormat(config, format)
aaxMixerSetSetup(config, AAX_FORMAT, format)
  • The format parameter is an enumerated type aaxFormat.
    Only the System Native formats are supported by the mixer.
Set the format of the audio that gets send to the backend driver. 
aaxMixerSetMonoSources(sources)
aaxMixerSetSetup(NULL, AAX_MONO_EMITTERS, sources)
Reserve a number of mono emitters for the current session.
aaxMixerSetStereoSources(sources)
aaxMixerSetSetup(NULL, AAX_STEREO_EMITTERS, sources)
Reserve a number of stereo emitters for the current session.
