Velocity

int aaxSensorSetVelocity(aaxConfig config, const aaxVec3f velocity)
int aaxEmitterSetVelocity(aaxEmitter emitter, const aaxVec3f velocity)
int aaxAudioFrameSetVelocity(aaxFrame frame, const aaxVec3f velocity)
Set the velocity vector.

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • Depending on the emitters position mode the velocity vector of the emitter should either be absolute, or relative to the sensors velocity vector.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle, a NULL pointer for the velocity parameter or a velocity vector that contains one ore more NaN's. 


int aaxSensorGetVelocity(const aaxConfig config, aaxVec3f velocity)
int aaxEmitterGetVelocity(const aaxEmitter emitter, aaxVec3f velocity)
int aaxAudioFrameGetVelocity(const aaxFrame frame, aaxVec3f velocity)
Stores the velocity vector in the velocity parameter.

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • Depending on the position mode the velocity vector of the emitter will either be absolute, or relative to the sensors velocity vector.
Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle or a NULL pointer for the velocity parameter.
