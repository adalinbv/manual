Buffer Offset

int aaxEmitterSetOffset(aaxEmitter emitter, unsigned long offset, enum aaxType type)
Set the offset from the start of the first buffer in the buffer stack.

  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • The offset parameter defines the number of units offset.
  • The type parameter is an enumerated type aaxType and describes the unit-type for the buffer offset.

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxEmitterSetOffsetSec(aaxEmitter emitter, float offset)
Set the offset in seconds from the start of the first buffer in the buffer stack.

  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • The offset parameter defines the number of seconds offset.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed handle or an incorrect type parameter.
