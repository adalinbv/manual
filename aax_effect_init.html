Effect initialization

aaxEffect aaxEffectCreate(aaxConfig config, enum aaxEffectType type)
Create a new effect instance for a specific type of effect.

  • The config parameter is the handle as returned by aaxDriverOpen.
  • The type parameter is an enumerated type aaxEffectType.
  • Assigning 3D effects to a stereo emitter or audio-frame has no effect.
  • Stereo effects can be assigned to emitters in any mode.

Return a newly created effect handle if successful or NULL otherwise.

Reasons for failure are in incorrect or already destroyed configuration handle or an incompatible type parameter or insufficient memory to allocate the effect handle. 

bool aaxEffectDestroy(aaxEffect effect)
Destroy an effect handle.

  • The effect parameter is the handle as returned by aaxEffectCreate or one of the class getter functions.

Return true if successful, false otherwise.

Reasons for failure are an incorrect or already destroyed handle.
