Delay Effect

The Delay effect feeds a delayed copy of the signal back to the undelayed signal.
Delay effects range from a subtle echo effect to a pronounced blending of previous sounds with new sounds.

AeonWave 3.9.2 adds support for the delay effect.

AeonWave 3.12.4 (version 1.04 of the effect) adds support for negative (phase shifted) levels and levels higher than 1.0 where the delayed signal is louder than the direct signal.

AeonWave 3.12.4 allows assigning both the delay effect and either phasing, chorus or flanging simultaneously.

The Delay effect is supported by the Mixer and Audio-Frames

Delay Effect state options
Specify a waveform type or envelope-following to enable the flanging effect or AAX_FALSE to disable it.
Timed-transition is a one time action from a start value to an end value which takes (1/lfo-frequency) seconds to complete.
Envelope following calculates the current amplitude and sets the value based on that with an update speed of (1/lfo-frequency) seconds. A low value for lfo-frequency therefore means it takes longer for the envelope following to reach the next value.
AAX_DELAY_EFFECT
