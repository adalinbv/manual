Delay Effect

The Delay effect feeds a delayed copy of the signal back to the undelayed signal.
Delay effects range from a subtle echo effect to a pronounced blending of previous sounds with new sounds.

Only one delay effect can be used at an object (mixer, emitter, audio-frame) at any given time: either Phasing, Chorus, Flanging or Delay.

AeonWave 3.9.2 adds support for the delay effect.

The Delay effect is supported by the Mixer and Audio-Frames
Delay Effect parameters
The effect uses only one effect slot.

STEREO

p0

Gain factor for the delayed signal.
AAX_LINEAR, AAX_LOGARITHMIC

Between 0.0 and 1.0
p1

Frequency of the low-frequency oscillator.
Between 0.01 and 10 Hz.
p2
AAX_LFO_DEPTH
Depth of the low-frequency oscillator.
Between 0.0 and 1.0, set to 0.0 for a constant offset.
p3
AAX_LFO_OFFSET
Offset position of the low-frequency oscillator.
Between 0.0 (10 milliseconds) and 1.0 (60 milliseconds)
Slot 1
p0
AAX_DELAY_CUTOFF_FREQUENCY
Cut-off frequency for the delayed signal in Hz.
Between 20 Hz and 22050 Hz
p1
AAX_DELAY_CUTOFF_FREQUENCY_HF
Higher cut-off frequency for the delayed signal in Hz.
Between 20 Hz and 22050 Hz
p2
AAX_FEEDBACK_GAIN
Gain of the feedback signal.
Between 0.0 and 1.0
p3
AAX_DELAY_RESONANCE
Resonance factor of the frequency filter.
Between 0.1 en 100.0, defaults to 1.0
Delay Effect state options
Specify AAX_TRUE to enable the delay effect or AAX_FALSE to disable it.
