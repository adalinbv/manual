Delay Effect

The Delay effect feeds a delayed copy of the signal back to the undelayed signal.
Delay effects range from a subtle echo effect to a pronounced blending of previous sounds with new sounds.

Only one delay effect can be used at an object (mixer, emitter, audio-frame) at any given time: either Phasing, Chorus, Flanging or Delay.

AeonWave 3.9.2 adds support for the delay effect.

The Delay effect is supported by the Mixer and Audio-Frames

Delay Effect state options
Specify AAX_TRUE to enable the delay effect or AAX_FALSE to disable it.
