Initialization

aaxAudioFrame aaxAudioFrameCreate(aaxConfig config)
Create a new AudioFrame.

  • The config parameter is the handle as returned by aaxDriverOpen.

Return a new audio-frame handle or NULL in case of an error.

Reasons for failure are an incorrect or already destroyed config handle or insufficient memory to allocate the new audio-frame handle. 

int aaxAudioFrameDestroy(aaxAudioFrame frame)
Destroy the frame and all it's associated memory allocations.

  • The frame handle can not be used after calling this function.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed handle or trying to destroy a registered audio frame. 

int aaxAudioFrameAddBuffer(aaxAudioFrame frame, aaxBuffer buffer)
Adds an audio buffer to the frame.

  • Only AAXS buffers containing filter and/or effect information are supported.
  • If a new buffer is added all previous defined filters and effects might be erased depending on the buffer specification.
  • If the buffer parameter is NULL or a non existent buffer the program shall continue without a problem.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed frame or buffer handle, adding a buffer which has no AAXS data assigned to it or insufficient memory to add the buffer to the audio-frame.
