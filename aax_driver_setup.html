Change Setup parameters

const char* aaxDriverGetSetup(const aaxConfig config, enum aaxSetupType type)

  • The config parameter is the handle as returned by aaxDriverOpen.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The buffer parameter is the handle as returned by aaxBufferCreate.
  • The type parameter specifies which setup parameter to return.

Return the current setup of the class as an enumerated type aaxSetupType or AAX_FALSE if a error occurred.

Reasons for failure are an incorrect or already destroyed class handle or an improper type parameter.

Supported values for the type parameter
AAX_NAME_STRING
Get the name of this library.
AAX_VERSION_STRING
Get the version information of the library.
AAX_RENDERER_STRING
Get the renderer name for this driver backend.
AAX_VENDOR_STRING
Get the vendor name for this driver backend.
AAX_SHARED_DATA_DIR
Get the directory where AeonWave stores shared data like preset files.
AAX_MUSIC_GENRE_STRING
Get the name of the music genre or NULL if unavailable.
AAX_MUSIC_PERFORMER_STRING
Get the name of the music performer or NULL if unavailable.
AAX_MUSIC_PERFORMER_UPDATE
Get the name of the music performer of the current track or NULL if it wasn't changed since the last request.
AAX_TRACK_TITLE_STRING
Get the name of the title or NULL if unavailable.
AAX_TRACK_TITLE_UPDATE
Get the name of the title of the current track or NULL if it wasn't changed since the last request..
AAX_TRACK_NUMBER_STRING
Get the track number or NULL if unavailable.
AAX_ALBUM_NAME_STRING
Get the album name or NULL if unavailable.
AAX_RELEASE_DATE_STRING
Get the release date or NULL if unavailable.
AAX_SONG_COPYRIGHT_STRING
Get the copyright holder or NULL if unavailable.
AAX_SONG_COMPOSER_STRING
Get the name of the composer(s) or NULL if unavailable.
AAX_SONG_COMMENT_STRING
Get the extra comments or NULL if unavailable.
AAX_ORIGINAL_PERFORMER_STRING
Get the name of the original performer or NULL if unavailable.
AAX_WEBSITE_STRING
Get the website address of the performer or NULL if unavailable.
