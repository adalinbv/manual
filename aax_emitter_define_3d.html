Emitter defines

None of these defined functions have any effect unless the emitter is registered to either the mixer or to an audio-frame. They can however be useful to set the emitter in a predefined state prior to registration. 
aaxEmitterSetIdentityMatrix(emitter)
Load the Identity matrix as the emitter position-and-orientation matrix.
aaxEmitterSetAudioCone(emitter, inner_angle, outer_angle, outer_gain)
Set the audio cone inner-angle and outer-angle in radians and linear outer-gain using the directional-filter.
If any of the inner-angle, outer-angle or outer-gain parameters is NaN it will be ignored. 
aaxEmitterSetDistanceModel(emitter, model)
  • the model parameter is an enumerated type aaxDistanceModel.
Set the preferred distance mode for distance attenuation using the distance-filter. 
aaxEmitterSetRolloffFactor(emitter, roll-off)
Set the roll-off factor for distance attenuation using the distance-filter. 
aaxEmitterSetReferenceDistance(emitter, distance)
Set the reference distance for distance attenuation using the distance-filter.
The reference distance is the distance from which the sound has been recorded. 
aaxEmitterSetMaxDistance(emitter, max_distance)
Set the maximum distance for distance attenuation using the distance-filter.
Depending on the distance-mode sounds beyond the maximum distance will be clamped, processed or culled. 
aaxEmitterEnableFrequencyFilter(emitter, enable)
Enable or disable the frequency-filter. 
aaxEmitterSetFrequencyFilter(emitter, cutoff, lf_gain, hf_gain)
Set the frequency filters cut-off frequency and linear low-frequency and high-frequency gain.
