Filter initialization

aaxFilter aaxFilterCreate(aaxConfig config, enum aaxFilterType type)
Create a new filter instance for a specific type of filter.

  • The config parameter is the handle as returned by aaxDriverOpen.
  • The type parameter is an enumerated type aaxFilterType.
  • Assigning 3D filters to a stereo emitter or audio-frame has no effect.
  • Stereo filters can be assigned to emitters in any mode.

Return a newly created filter handle if successful or NULL otherwise.

Reasons for failure are in incorrect or already destroyed configuration handle or an incompatible type parameter or insufficient memory to allocate the filter handle. 

bool aaxFilterDestroy(aaxFilter filter)
Destroy a filter handle.

  • The filter parameter is the handle as returned by aaxFilterCreate or one of the class getter functions.

Return true if successful, false otherwise.

Reasons for failure are an incorrect or already destroyed handle.
