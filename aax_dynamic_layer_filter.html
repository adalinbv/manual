Dynamic Layer Filter

The Dynamic Layer filter dynamically mixes two sound layers from an AAXS file.

If the combined Offset parameter and Depth parameter exceeds 1.0 then the depth of the low frequency oscillator is automatically clamped.

The Dynamic Layer filter is supported by Emitters.
Dynamic Layer Filter parameters
The filter uses only one filter slot.
AAX_DYNAMIC_LAYER_FILTER
MONO
p0
AAX_INITIAL_DELAY
Delay before the filter kicks in.
Between 0.0 and 10.0 seconds.
p1
AAX_LFO_FREQUENCY
Frequency of the low-frequency oscillator for mixing.
Response rate in Hertz (1/seconds) when using Envelope Following mode.
Between 0.01 and 50 Hz.
p2
AAX_LFO_MAX
Maximum offset value of the low-frequency oscillator.
Between 0.0 and 1.0.
p3
AAX_LFO_MIN
Minimum offset value of the low-frequency oscillator.
Between 0.0 and 1.0
Dynamic Layer Filter state options
Specify a waveform type or envelope-following to enable the dynamic filter or false to disable it.
AAX_DYNAMIC_LAYER_FILTER
Valid waveform types are:
AAX_CONSTANT_VALUE (or true)
AAX_TRIANGLE_WAVE
AAX_SINE_WAVE
AAX_SQUARE_WAVE
AAX_SAWTOOTH_WAVE
AAX_ENVELOPE_FOLLOW
AAX_INVERSE_TRIANGLE_WAVE
AAX_INVERSE_SINE_WAVE
AAX_INVERSE_SQUARE_WAVE
AAX_INVERSE_SAWTOOTH_WAVE
AAX_INVERSE_ENVELOPE_FOLLOW

All waveforms may be combined with AAX_ENVELOPE_FOLLOW_LOG or AAX_INVERSE_ENVELOPE_FOLLOW_LOG to create exponential behaviour, e.g. AAX_ENVELOPE_FOLLOW|AAX_ENVELOPE_FOLLOW_LOG
