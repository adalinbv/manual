AeonWave AAXS Schema Reference

Data structure guide for AeonWave audio synthesis formats.

Root Structure

The root element is <aeonwave>. It contains four primary blocks:

  • info: Metadata
  • sound: Synthesis
  • emitter (Optional): Spatial/Output
  • audioframe (Optional): Per-frame Overrides

2. Sound & Layering Logic

Global Attributes: frequency (Required), gain, duration, voices, spread, phasing, mode.

Layers (max. 2)

Each layer contains one or multiple:

  • Waveforms: Include properties like src (source type), processing (mix/add/modulate), ratio, pitch, phase, staticity, and random.
  • Filters (optional): Various types.
  • Effects (optional): Various effects.
