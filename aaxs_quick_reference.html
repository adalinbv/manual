2. Synthesis (sound)


Up to 2 layers are supported, which are mixed using the dynamic or timed layer filters.
Each layer contains (one or more) waveform and (optional) filter, and effect in an arbitrary sequence.

3. Modulation & Timeline

Both emitter and audioframe act as containers for signal chains, but serve different purposes in the XML structure.

3.1 emitter

Role: The source of modulation (e.g., an LFO or Envelope).

AttributeType
pandecimal
gaindecimal
loopingboolean

3.2 audioframe

Role: Timeline/Sequencing instructions for audio.

AttributeType
gaindecimal
pandecimal
View Shared Component Structure

Both types follow this sequence (repeating):

  • filter (Optional, 0 to multiple)
  • effect (Optional, 0 to multiple)

Note: Each filter/effect follows the sharedSlotGroup rules.
