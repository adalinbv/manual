Reverb Effect

Reverberation or Reverb is an effect that uses a large number of delayed copies of the original signal to give an impression of the surrounding environment. As a result the audio signal will remain present after it was produced, sometimes for a long period of time.

Rooms with furniture, carpet and curtains reflect lower frequencies but tend to absorb higher frequencies more quickly while empty rooms with concrete surfaces reflect the full frequency spectrum.

The initial delay between the direct path of the sound and the arrival of the first reflections give a good first indication of the size of the room.
Set the Delay Depth parameters to low for small rooms and high for large rooms.

Set the Decay Level parameter high for highly reflective rooms and set a low decay level to dampen the reflections rather quickly. Set it to zero to only process the first order reflections.

Set the Decay Depth parameter low for a small room and set high for a large industrial complex or a stadium.
Very high values for the Decay Depth parameter are perceived as echo.

AeonWave version 3.1 (and version 1.1 of the reverb effect) adds the option to add reverb to audio frames to support indoor sound propagation but only with a limited decay depth of 210 ms. This enables simulations up to large rooms. All reflections beyond that will simply be ignored.

AeonWave version 3.1 adds support for occlusion by defining a 3D obstruction (or 3D cavity) at the same position, and with the same orientation, as the audio-frame. The obstruction/cavity could influence the gain of the emitter based on the relative position of the emitter, the audio-frame(s) and listener.

AeonWave does not define the units of measurement for size, the calculations are scale invariant. Developers have to make sure that all distances and sizes used by AeonWave use the same unit.

Reverb is supported by the Scenery and by Audio-Frames.
Occlusion is only supported by Audio-frames.
Reverb parameters
The effect uses one or two effect slots.
AAX_REVERB_EFFECT
STEREO|3D
Slot 0
p0
AAX_CUTOFF_FREQUENCY
Alters the frequency response of the room by setting a Low Pass frequency filter cut off frequency.
Between 50Hz and 22kHz.
p1
AAX_DELAY_DEPTH
The initial delay between the direct path of the sound and the arrival of the first reflections.
Between 0.0 and 0.07 (70 ms).
p2
AAX_DECAY_LEVEL
Sets the gain for the reflections.
Between 0.0 and 1.0
p3
AAX_DECAY_DEPTH
Defines how much time it takes for a reflection to reach the listener again.
Between 0.0 and 1.0 (1 second).
Slot 1
p0
AAX_OBJECT_WIDTH
The width of the occlusion object. The width is specified in the x-direction.
p1
AAX_OBJECT_HEIGHT
The height of the occlusion object. The height is specified in the y-direction.
p2
AAX_OBJECT_DEPTH
The depth of the occlusion object. The depth is specified in the z-direction.
p3
AAX_OBJECT_DENSITY_FACTOR
The density of the occlusion object.
Between 0.0 and 1.0, where 1.0 means silence when the object is directly between the emitter and sensor.
Reverb state options
Specify AAX_TRUE to enable reverb or AAX_FALSE to disable it.
For occlusion it is required to set the state to AAX_TRUE (the dimensions define an obstruction) or AAX_INVERSE (the dimensions define a hole).
