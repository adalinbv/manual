Buffer defines

aaxBufferSetFrequency(buffer, frequency)
aaxBufferSetSetup(buffer, AAX_FREQUENCY, frequency)
Set the sample frequency of the audio buffer. 
aaxBufferSetNoTracks(buffer, tracks)
aaxBufferSetSetup(buffer, AAX_TRACKS, tracks)
Set the number of tracks in the audio buffer. 
aaxBufferSetLoopPoints(buffer, loop_start, loop_end)
Set the loop-start and loop-end points as samples from the start of the buffer.
Setting the start point beyond the end point results in an error. 
aaxBufferSetWaveform(buffer, frequency, type)
aaxBufferProcessWaveform(buffer, frequency, type, 1.0, AAX_OVERWRITE)
Fill the buffer with a newly generated waveform of a given frequency and waveform-type by overwriting existing data. 
aaxBufferAddWaveform(buffer, frequency, type, gain)
aaxBufferProcessWaveform(buffer, frequency, type, gain, AAX_ADD)
Add a newly generated waveform of a given frequency and waveform-type and with the specified gain to the current data in the buffer. 
aaxBufferMixWaveform(buffer, frequency, type, ratio)
aaxBufferProcessWaveform(buffer, frequency, type, ratio, AAX_MIX)
Mix a newly generated waveform of a given frequency and waveform-type with the current data in the buffer at the specified mixing ratio. 
aaxBufferRingmodulateWaveform(buffer, freq, type, depth)
aaxBufferProcessWaveform(buffer, freq, type, ratio, AAX_RINGMODULATE)
Ring-modulate the current data using a newly generated waveform of a given frequency and waveform-type at the specified modulation depth. 


aaxBufferIsValid(handle)
aaxIsValid(handle, AAX_BUFFER)
Return AAX_TRUE if the handle is a valid buffer handle, AAX_FALSE otherwise. 
aaxBufferGetFrequency(buffer)
aaxBufferGetSetup(buffer, AAX_FREQUENCY)
Return the sample frequency of the audio buffer. 
aaxBufferGetFormat(buffer)
aaxBufferGetSetup(emitter, AAX_FORMAT)
Return the format of the audio buffer. 
aaxBufferGetNoTracks(buffer)
aaxBufferGetSetup(buffer, AAX_TRACKS)
Return the number of tracks in the audio buffer. 
aaxBufferGetSize(buffer)
Return the total buffer size in bytes of all tracks in the audio buffer. 
aaxBufferGetNoSamples(buffer)
aaxBufferGetSetup(buffer, AAX_NO_SAMPLES)
Return the number of samples per track in the audio buffer. 
aaxBufferGetBytesPerSample(buffer)
aaxGetBytesPerSample(aaxBufferGetSetup(buffer, AAX_FORMAT))
Return the number of bytes per sample in the audio buffer.
