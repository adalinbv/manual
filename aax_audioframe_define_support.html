AudioFrame defines

aaxAudioFrameIsValid(handle)
aaxIsValid(handle, AAX_CONFIG)
Return AAX_TRUE if the handle is a valid audio-frame handle, AAX_FALSE otherwise. 
aaxAudioFrameGetGain(frame)
Return the linear gain value of this frame using the volume-filter. 
aaxAudioFrameGetPitch(frame)
Return the pitch factor of this frame using the pitch-effect. 
aaxAudioFrameSetSoundVelocity(frame)
Return the sound velocity setting of the frame using the velocity-effect.
aaxAudioFrameSetDopplerFactor(frame)
Return the Doppler factor of the frame using the velocity-effect.
