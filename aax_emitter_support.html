Support

float aaxEmitterGetOffsetSec(const aaxEmitter emitter)
  • The emitter parameter is the handle as returned by aaxEmitterCreate.

Return the actual playing time in seconds since the emitter started playback or false in case of an error.

Reasons for failure are an incorrect or already destroyed handle.

const aaxBuffer aaxEmitterGetBufferByPos(const aaxEmitter emitter, unsigned int pos,
                                           int copy)
  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • The pos parameter defines which buffer number to return from the buffer queue.
  • The copy parameter defines if a buffer should be copied or just referenced.
    A copied buffer needs to be destroyed by aaxBufferDestroy but a referenced buffer may become invalid when the last reference to it has been destroyed.

Return the buffer at the requested position in the buffer stack if successful or NULL otherwise. The buffer will not be removed from the queue.

Reasons for failure are an incorrect or already destroyed handle or an incorrect position parameter. 

unsigned int aaxEmitterGetNoBuffers(const aaxEmitter emitter, enum aaxState state)
  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • The state parameter is an enumerated type aaxState that describes which type of buffer to get information for.

Return the number of buffers currently in the buffer stack.

Supported values for the state parameter
AAX_PLAYING
Return the number of buffers that still need to play until the buffer stack of the emitter is processed.
AAX_PROCESSED
Return the number of buffers that have been processed during playback of an emitter.
Since these buffers are already processed this number of buffers can be safely removed.
AAX_MAXIMUM
Return the total number of buffers in the buffer stack of the emitter.
