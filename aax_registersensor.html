Registering or De-registering

Emitters and audio-frames need to be registered before they get processed. Sensors that are opened in read-mode may be registered at either the mixer or an audio-frame, resulting in auto-streaming of the captured data.

It is possible to register exactly one sensor that was opened in write-mode for an audio-file at the mixer. This will allow the same audio data that is sent to the hardware to be streamed to the output file. 


bool aaxMixerRegisterEmitter(aaxConfig config, const aaxEmitter emitter)
bool aaxAudioFrameRegisterEmitter(aaxFrame frame, const aaxEmitter emitter)

Register an emitter at a mixer or audio-frame (sub-mixer).

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The emitter parameter is the handle as returned by aaxEmitterCreate.

Return true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed config, emitter or audio-frame handle or insufficient memory to create internal buffers.

bool aaxMixerRegisterAudioFrame(aaxConfig config, const aaxFrame frame)
bool aaxAudioFrameRegisterAudioFrame(aaxFrame frame_config, const aaxFrame frame)

Register an audio-frame at the mixer.
The audio-frame must be set to the AAX_PLAYING state before registering.

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame_config is the handle as returned by aaxAudioFrameCreate.
    Note: It is only valid to register audio-frames if the mixer is properly initialized.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.

Return true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed config or audio-frame handle, insufficient memory to create internal buffers or when trying to register an audio-frame that is not in the proper state.

bool aaxMixerRegisterSensor(aaxConfig config, const aaxConfig sensor)
bool aaxAudioFrameRegisterSensor(aaxFrame frame, const aaxConfig sensor)

Register a sensor (input device) at a mixer or audio-frame (sub-mixer).
The sensor will only get processed after it is set to the AAX_CAPTURING state.

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The sensor parameter is the handle as returned by aaxOpenDevice in AAX_MODE_READ mode.

Return true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed config, sensor or audio-frame handle or insufficient memory to create internal buffers.


bool aaxMixerDeregisterEmitter(aaxConfig config, const aaxEmitter emitter)
bool aaxAudioFrameDeregisterEmitter(aaxFrame frame, const aaxEmitter emitter)

De-register an emitter from a mixer or audio-frame (sub-mixer).

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The emitter parameter is the handle as returned by aaxEmitterCreate.

Return true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed config, emitter or audio-frame handle.

bool aaxMixerDeregisterAudioFrame(aaxConfig config, const aaxFrame frame)
bool aaxAudioFrameDeregisterAudioFrame(aaxFrame frame_config, const aaxFrame frame)

De-register an audio-frame from the mixer.

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame_config is the handle as returned by aaxAudioFrameCreate.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.

    Return true if successful or false otherwise.

    Reasons for failure are an incorrect or already destroyed config or audio-frame handle.

    bool aaxMixerDeregisterSensor(aaxConfig config, const aaxConfig sensor)
    bool aaxAudioFrameDeregisterSensor(aaxFrame frame, const aaxConfig sensor)

    De-register a sensor from a mixer or audio-frame (sub-mixer).

    • The config parameter is the handle as returned by aaxOpenDevice.
    • The frame parameter is the handle as returned by aaxAudioFrameCreate.
    • The sensor parameter is the handle as returned by aaxOpenDevice in AAX_MODE_READ mode.

    Return true if successful or false otherwise.

    Reasons for failure are an incorrect or already destroyed config, sensor or audio-frame handle.
