Position and Orientation

int aaxAudioFrameSetMatrix64(aaxFrame frame, const aaxMtx4d mtx)
Set the position-and-orientation matrix.

  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • Depending on the audio-frames position mode the position and orientation of the audio-frame should either be absolute, or relative to the sensors position and orientation.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle, a NULL pointer for the mtx parameter or a matrix that contains one ore more NaN's. 


int aaxAudioFrameGetMatrix64(const aaxConfig config, aaxMtx4d mtx)
Stores the position-and-orientation matrix int mtx.

  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • Depending on the audio-frames position mode the position and orientation of the audio-frame will either be absolute, or relative to the sensors position and orientation.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle or a NULL pointer for the mtx parameter.
