System Native Format Types in System Endianness
AAX_PCM8S
Native
Signed, uncompressed 8-bit PCM data.
AAX_PCM16S
Native
Signed, uncompressed 16-bit PCM data.
AAX_PCM24S
Native
Signed, uncompressed 24-bit PCM data in 32-bit samples.
AAX_PCM32S
Native
Signed, uncompressed 32-bit PCM data.
AAX_FLOAT
Native
32-bit uncompressed floating-point data. Range: -1.0 .. 1.0
AAX_DOUBLE
Native
64-bit uncompressed floating-point data. Range: -1.0 .. 1.0
AAX_MULAW
Native
8-bit µ-Law encoded, 16 to 8-bit compression by reducing the dynamic range.
AAX_ALAW
Native
8-bit A-Law encoded, 16 to 8-bit compression by reducing the dynamic range.
AAX_IMA4_ADPCM
Native
4-bit Differential PCM, 4-bit compressed using the differences between samples.
  • The block size can be set by calling aaxBufferSetup with the AAX_BLOCK_ALIGNMENT type specifier
    prior to calling aaxBufferSetData.
  • Interleaving is done by interleaving blocks (in contrast to MS WAV files which interleaves in 4 byte chunks).
AAX_PCM24S_PACKED
Signed, packed uncompressed 24-bit PCM data in 3-byte samples.

Unsigned Format Types in System Endianness
AAX_PCM8U
Unsigned, uncompressed 8-bit PCM data. 127 equals 0 Vtt.
AAX_PCM16U
Unsigned, uncompressed 16-bit PCM data. 32767 equals to 0Vtt.
AAX_PCM24U
Unsigned, uncompressed 24-bit PCM data in 32-bit samples. 8388607 equals to 0Vtt.
AAX_PCM32U
Unsigned, uncompressed 32-bit PCM data. 2147483647 equals to 0Vtt.
