Capture support

bool aaxSensorWaitForBuffer(aaxConfig config, float timeout)
bool aaxAudioFrameWaitForBuffer(aaxFrame frame, float timeout)
Wait for at least one new captured buffer to be available, or until timeout time has passed.

  • The config parameter is the handle as returned by aaxDriverOpen.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The timeout parameter specifies after how many seconds the function shall return anyhow.

Return true if a new buffer is available in the buffer stack or false otherwise.

Reasons for failure are an incorrect or already destroyed class handle or a timeout.
