Driver support

There are a number of functions available to be able to detect which backends are supported and which devices are available to;
  • return the number of backends.
  • get information about the backend at a given position.
  • return the number of devices for a given backend.
  • get the name of the device at a given position. 
aaxConfig aaxDriverGetByName(const char* name, enum aaxRenderMode mode)
  • the name parameter consists of the backend name, possibly followed by " on " and the device name.
    if no device name is specified the default will be used.
  • the mode parameter is an enumerated type aaxRenderMode.
Return the driver configuration for the requested backend name. 
unsigned aaxDriverGetCount(enum aaxRenderMode mode)
  • the mode parameter is an enumerated type aaxRenderMode.
Return the number of driver backends that this system supports for the specified mode. 
aaxConfig aaxDriverGetByPos(unsigned pos, enum aaxRenderMode mode)
  • the mode parameter is an enumerated type aaxRenderMode.
Return the driver configuration for the driver at the specified position and mode. 
unsigned aaxDriverGetDeviceCount(aaxConfig config, enum aaxRenderMode mode)
  • the mode parameter is an enumerated type aaxRenderMode.
Return the number of available devices that support the specified mode for this driver configuration. 
const char* aaxDriverGetDeviceNameByPos(aaxConfig config, unsigned pos, enum aaxRenderMode mode)
  • the pos parameter defines the position of the device to return.
  • the mode parameter is an enumerated type aaxRenderMode.
Return the device name at the specified position for the specified mode as a string constant. 
unsigned aaxDriverGetInterfaceCount(aaxConfig config, unsigned dev_pos, enum aaxRenderMode mode)
  • the dev_pos parameter is the position of the device of which to acquire the available interfaces.
  • the mode parameter is an enumerated type aaxRenderMode.
Return the number of available interfaces that support the specified mode for this driver configuration. 
const char* aaxDriverGetInterfaceNameByPos(aaxConfig config, usnigned dev_pos, unsigned pos,
                                             enum aaxRenderMode mode)
  • the dev_pos parameter defines the position of the device to get information from.
  • the pos parameter defines the position of the interface to return.
  • the mode parameter is an enumerated type aaxRenderMode.
Return the interface name at the specified position for the specified mode as a string constant. 
bool aaxDriverGetSupport(aaxConfig config, enum aaxRenderMode mode)
  • the mode parameter is an enumerated type aaxRenderMode.
Return true if the driver supports the requested mode, false otherwise.
