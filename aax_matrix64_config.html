64-bit Matrix Runtime Configuration

int aaxMatrix64SetIdentity(aaxMtx4d mtx)
Load the Identity matrix into mtx.

Return AAX_TRUE. 

int aaxMatrix64Translate(aaxMtx4d mtx, double dx, double dy, double dz)
Shift the position of the matrix by dx, dy and dz.

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrix64Rotate(aaxMtx4d mtx, double angle, double x, double y, double z)
Rotate the matrix by angle radians around the axis defined by (x, y, z).

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrix64Multiply(aaxMtx4d mtx, aaxMtx4d mtx2)
Multiply both matrices mtx and mtx2 and store the result in mtx

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrix64Inverse(aaxMtx4d mtx)
Calculate the inverse of the matrix and store the result in mtx.

Return AAX_TRUE if successful, AAX_FALSE otherwise.
