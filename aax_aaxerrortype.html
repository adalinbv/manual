enum aaxErrorType

AAX_ERROR_NONE
No Error
AAX_INVALID_DEVICE
The specified device is unavailable.
AAX_INVALID_SETUP
One or more of the setup parameters are unsupported by this device.
AAX_INVALID_HANDLE
The handle is invalid or might already be destroyed.
AAX_INVALID_STATE
Unable to finish the requested action due to the internal state of the class.
AAX_INVALID_ENUM
The supplied enumerated type is not supported.
AAX_INVALID_PARAMETER
The parsed parameter value is out of range.
AAX_INVALID_REFERENCE
Action on the requested type reference is impossible, most probably because it is not referenced (anymore).
AAX_INSUFFICIENT_RESOURCES
Insufficient resources to complete the task; this includes not enough memory to allocate an internal buffer.
AAX_TIMEOUT
The requested event timed out.
