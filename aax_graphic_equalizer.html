Graphic Equalizer

Implements a Graphic equalizer with 8 gain adjustable frequency bands.
For notch filtering use the parametric equalizer instead.

Gain values up to 10.0 (+20dB) are allowed but may result in undesirable sound artefacts;
A high gain value might cause the audio compressor to take action but a too high value might result in audio clipping which produces an unpleasant form of audio distortion.

Only supported by the Mixer.
Replaces the parametric equalizer if it was already defined.

Graphic Equalizer State options
Specify true to enable the graphic equalizer or false to disable it.
