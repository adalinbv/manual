Class AeonWave

Header file: <aax/aeonwave.hpp>
Namespace: aax

 

The AeonWave C++ API turns device enumeration into a simple nested while loop.

Each of drivers(), devices() and interfaces() functions will return a string constant referencing the associated name at the current position, and increment the position counter, until the last one is reached. In which case it returns 0 and the position counter is reset to 0.

After a successful run the process can therefore be repeated for a different rendering mode. 


   aax::AeonWave aax(AAX_MODE_WRITE_STEREO);

   std::cout << "Devices that support capture:" << std::endl;
   while (const char* d = aax.drivers(AAX_MODE_READ)) {
       while (const char* r = aax.devices()) {
           while (const char* i = aax.interfaces()) {
               std::cout << " '" << d;
               if (*r) std::cout << " on " << r;
               if (*i) std::cout << ": " << i ;
               std::cout << "'" << std::endl;
           }
       }
    }
    std::cout << std::endl;

    std::cout << "Devices that support playback:" << std::endl;
