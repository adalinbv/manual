Position and Orientation

int aaxEmitterSetMatrix64(aaxEmitter emitter, const aaxMtx4d mtx)
Set the position-and-orientation matrix.

  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • Depending on the emitters position mode the position and orientation of the emitter should either be absolute, or relative to the sensors position and orientation.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle, a NULL pointer for the mtx parameter or a matrix that contains one ore more NaN's. 


int aaxEmitterGetMatrix64(const aaxEmitter emitter, aaxMtx4d mtx)
Stores the position-and-orientation matrix int mtx.

  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • Depending on the emitters position mode the position and orientation of the emitter will either be absolute, or relative to the sensors position and orientation.

Return AAX_TRUE if successful, AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle or a NULL pointer for the mtx parameter.
