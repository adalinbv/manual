Capture control

aaxBuffer aaxSensorGetBuffer(const aaxConfig config)
aaxBuffer aaxAudioFrameGetBuffer(const aaxFrame frame)
Return and remove the first available buffer from the buffer stack or NULL of none are available.

  • The config parameter is the handle as returned by aaxOpenDevice.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • It is also possible to wait for the next buffer by using the aaxSensorWaitForBuffer function.
  • Another way to test if there is at least one buffer available in the buffer stack is to use the aaxSensorGetOffset function and detect if it returns a value greater than zero.
