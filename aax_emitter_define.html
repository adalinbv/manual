Emitter defines

None of these defined functions have any effect unless the emitter is registered to either the mixer or to an audio-frame. They can however be useful to set the emitter in a predefined state prior to registration. 
aaxEmitterStart(emitter)
aaxEmitterSetState(emitter, AAX_PLAYING)
Schedule the playback of the emitter. 
aaxEmitterSuspend(emitter)
aaxEmitterSetState(emitter, AAX_SUSPENDED)
Temporarily suspend playback of the emitter.
The configuration of the emitter will not be affected by this call. 
aaxEmitterResume(emitter)
aaxEmitterSetState(emitter, AAX_PLAYING)
Resume playback of the emitter after suspending.
aaxEmitterStop(emitter)
aaxEmitterSetState(emitter, AAX_STOPPED)
Permanently stop playback of this emitter.
The configuration of the emitter will be set to initialized but the setup will not be affected. 
aaxEmitterRewind(emitter)
aaxEmitterSetState(emitter, AAX_INITIAL)
Rewind the playback position to the start of the first buffer in the emitters buffer stack.
aaxEmitterSetOffsetBytes(emitter, offset)
aaxEmitterSetOffset(emitter, offset, AAX_BYTES)
Set the playback position to the real sample specified by the offset in bytes from the start of the first buffer in the emitters buffer stack. 
aaxEmitterSetOffsetFrames(emitter,offset)
aaxEmitterSetOffset(emitter, offset, AAX_FRAMES)
Set the playback position to the specified audio frame from the start of the first buffer in the emitters buffer stack. 
aaxEmitterSetOffsetSamples(emitter, offset)
aaxEmitterSetOffset(emitter, offset, AAX_SAMPLES)
Set the playback position to the specified number of samples from the start of the first buffer in the emitters buffer stack. 
aaxEmitterSetGain(emitter, gain)
Set the emitters linear gain using the volume-filter. 
aaxEmitterSetGainMinMax(emitter, min, max)
Set the emitters linear minimum and maximum allowed gain using the volume-filter. 
aaxEmitterSetPitch(emitter, pitch)
Set the emitters pitch factor using the pitch-effect. 
aaxEmitterSetBufferTrack(emitter, track)
aaxEmitterSetMode(emitter, AAX_BUFFER_TRACK, track)
Define which track number of a multi-track buffer to use for playback when in 3d mode. 
aaxEmitterEnableLooping(emitter, looping)
aaxEmitterSetMode(emitter, AAX_LOOPING, looping)
Set or unset buffer sample looping.
The loop points are defined by the associated buffer.
