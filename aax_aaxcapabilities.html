enum aaxCapabilities

Use the following bitmasks to return the information from the return value:
AAX_CPU_CORES
The number of physical CPU cores. Increase by one to get the real number of cores.
The maximum number of reported CPU cores is 64.
AAX_64BIT
Set to 0 on 32-bit systems and to 1 otherwise.
AAX_SIMD
Set to 1 for 32-bit SIMD support
AAX_SIMD64
Set to 1 for 64-bit SIMD support
AAX_RENDER_MODE
The render mode for AAXS encoded instruments:
  • AAX_RENDER_NORMAL: The default intricate rendering mode.
  • AAX_RENDER_SYNTHESIZER: An 80's FM Synthesizer style mode.
  • AAX_RENDER_ARCADE: A 70's Arcade Game Console style mode.
The system capabilities information is intended for the developer to determine a strategy for not overloading the system resources. For instance by limiting the number of emitters or by omitting certain types of filters or effects on system with a low number of resources.
