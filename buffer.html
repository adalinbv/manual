Class Buffer

Header file: <aax/aeonwave>
Namespace: aeonwave

 

The Buffer class encapsulates the Buffer functions.

Buffers are in essence containers of audio data and sample information Because of the static nature of audio buffers they can be shared among multiple registered emitters at the same time.

An internal waveform generator is available to fill audio buffers on-demand.

It is recommended to use the buffer management code from the AeonWave class to manage buffers.
This code can also be used to create buffers from local or remote files. By using AAXS files as buffer objects it is easy to apply filters and effects to emitters and audio-frames and to generate sounds using the waveform generator.
