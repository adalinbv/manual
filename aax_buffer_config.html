Buffer runtime configuration

bool aaxBufferSetData(aaxBuffer buffer, const void* data)
Store new sample data in the buffer.

  • The buffer parameter is the handle as returned by aaxBufferCreate.
  • The data parameter is a pointer to a memory area that holds the sound data in the format specified when creating the buffer using aaxBufferCreate.
  • When the sound buffer contains multiple tracks they should be interleaved according to the compression type specification.
  • The sound data will be copied for internal use and the buffer pointed to by the data parameter should be freed by the caller.

Return true if successful, false otherwise.

Reasons for failure are an incorrect or already destroyed buffer handle or the data parameter being NULL. 


void** aaxBufferGetData(const aaxBuffer buffer)
  • The buffer parameter is the handle as returned by aaxBufferCreate.
  • The returned data is stored interleaved and in the format associated with the buffer.
  • The sound data area is guaranteed to be 16-byte aligned on systems that support SSE.
  • Freeing the returned pointer will free all returned data.

Returns a pointer to a pointer to a memory area containing a copy of the sample data of the audio buffer or NULL in case of an error.

Reasons for failure are an incorrect or already destroyed buffer handle or insufficient memory to create the sound buffer.
