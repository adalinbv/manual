Filter initialization

int aaxFilterSetSlot(aaxFilter filter, unsigned slot, int ptype,
                                              float p1, float p2, float p3, float p4)
Same as aaxFilterSetSlotParams except that the fields of the values parameter have to be specified independently.

  • p1 defines values[0], p2 defines values[1], p3 defines values[2] and p4 defines values[3]. 
int aaxFilterSetSlotParams(aaxFilter filter, unsigned slot, int ptype, aaxVec4f values)
Set the values of a parameter-slot for a filter.
Most filters need just one parameter-slot but some require more slots to be set properly.

  • The filter parameter is the handle as returned by aaxFilterCreate or one of the class getter functions.
  • The slot parameter defines which slot number to alter.
    Slot numbers start at zero.
  • The ptype parameter specifies the unit of the values parameter and is type specific.
    See enum aaxType for more information.
  • The values parameter is an array of four float type parameters that represents the filter-type specific filter parameters.
    See aaxFilterType for more information.
  • If one or more of the fields of the values parameter is NaN it's value will not bet set.
Return AAX_TRUE if successful or AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed filter handle or an improper slot number. 

int aaxFilterSetParam(aaxFilter filter, int param_no, int ptype, float value)
Set the value of the specified parameter of the filter.

  • The filter parameter is the handle as returned by aaxFilterCreate or one of the class getter functions.
  • The param_no parameter defines which one of the parameters to alter.
    See enum aaxParameter for more information.
  • The ptype parameter specifies the unit of the parameter and is type specific.
    See enum aaxType for more information.
  • If the value parameter is NaN the parameter value will not bet set.

Return AAX_TRUE if successful or AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed filter handle, an incorrect param_no value or the value parameter is a NaN. 

int aaxFilterSetState(aaxFilter filter, int state)
Change the state of the filter.

  • The filter parameter is the handle as returned by aaxFilterCreate or one of the class getter functions.
  • Possible values for the state parameter differ between filter types
    See aaxFilterState for more information.

Return AAX_TRUE if successful or AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed filter handle.
