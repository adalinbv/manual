AudioFrame defines

Audio-frames that are registered to the mixer will behave differently than audio-frames that are not registered. The output of registered frames will automatically be added to the final mixer stage of the mixer. The output of non-registered frames will need to be fetched by the calling program. Failure to do so might result in memory overflow. 
aaxAudioFrameStart(frame)
aaxAudioFrameSetState(frame, AAX_PLAYING)
Start playback of emitters registered to the audio-frame.
An audio-frame will always start but depending on whether it is registered to the mixer or not it might be necessary for the calling program to regularly check for available buffers to prevent memory overflow. 
aaxAudioFrameSuspend(frame)
aaxAudioFrameSetState(frame, AAX_SUSPENDED)
Temporarily suspend playback of the audio-frame.
The frame configuration will not be affected by this call. 
aaxAudioFrameResume(frame)
aaxAudioFrameSetState(frame, AAX_PLAYING)
Resume playback of the frame after suspending. 
aaxAudioFrameStop(frame)
aaxAudioFrameSetState(frame, AAX_STOPPED)
Permanently stop playback of this frame.
The configuration of the frame will be set to initialized but the setup will not be affected. 
aaxAudioFrameRewind(frame)
aaxAudioFrameSetState(frame, AAX_INITIAL)
Rewind the playback position to the start of the first buffer in the frames buffer stack. 
aaxAudioFrameSetOffsetBytes(frame, offset)
aaxAudioFrameSetOffset(frame, offset, AAX_BYTES)
Set the playback position to the real sample specified by the offset in bytes from the start of the first buffer in the frames buffer stack. 
aaxAudioFrameSetOffsetFrames(frame, offset)
aaxAudioFrameSetOffset(frame, offset, AAX_FRAMES)
Set the playback position to the specified audio frame from the start of the first buffer in the frames buffer stack. 
aaxAudioFrameSetOffsetSamples(frame, offset)
aaxAudioFrameSetOffset(frame, offset, AAX_SAMPLES)
Set the playback position to the specified number of samples from the start of the first buffer in the frames buffer stack. 
aaxAudioFrameSetGain(frame, gain)
Set the linear gain of the audio-frame using the volume-filter.
This will act as a master gain for all registered emitters. 
aaxAudioFrameSetPitch(frame, pitch)
Set the pitch factor of the frame using the pitch-effect.
This will act as pitch shifter for all registered emitters. 
aaxAudioFrameSetIdentityMatrix(frame)
Load the Identity matrix as the frame position-and-orientation matrix.
aaxAudioFrameSetSoundVelocity(frame, speed)
Set the sound velocity of the frame using the velocity-effect. 
aaxAudioFrameSetDopplerFactor(frame, Doppler)
Set the Doppler factor of the frame using the velocity-effect. 
aaxAudioFrameSetDistanceModel(frame, model)
Set the preferred distance mode for distance attenuation using the distance-filter. 
aaxAudioFrameEnableFrequencyFilter(frame, enable)
Enable or disable the frequency-filter. 
aaxAudioFrameSetFrequencyFilter(frame, cutoff, lf_gain, hf_gain)
Set the frequency filters cut-off frequency and linear low-frequency and high-frequency gain.
