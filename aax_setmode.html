Modifying the Playback mode

int aaxEmitterSetMode(aaxEmitter emitter, enum aaxModeType type, int mode)
int aaxAudioFrameSetMode(aaxFrame frame, enum aaxModeType type, int mode)

Alter the mode of operation.

  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The emitter parameter is the handle as returned by aaxEmitterCreate.
  • The type parameter is an enumerated type and specifies which mode to change.
  • The mode parameter is type parameter specific (see aaxModeType for more information).

Return AAX_TRUE if successful or AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle, an incompatible type parameter or an invalid mode value. 


int aaxEmitterGetMode(const aaxEmitter emitter, enum aaxModeType type)

Return current mode settings.

  • The emitter parameter is the handle as returned by aaxEmitterCreate
  • The type parameter specifies which mode parameter to return.

Return the value of the requested mode type or AAX_FALSE if a error occurred.

Reasons for failure are an incorrect or already destroyed class handle or an incompatible type parameter.
