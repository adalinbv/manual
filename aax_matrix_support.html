Matrix support

int aaxMatrixGetOrientation(aaxMtx4f mtx, aaxVec3f pos, aaxVec3f at, aaxVec3f up)
Get either the position, at vector and/or up vector from the provided matrix.

  • If one or more of the pos, at or up parameters are NULL it's value will not be set but the program shall continue without a problem.

Return true if successful, false otherwise.
