Matrix Runtime Configuration

int aaxMatrixSetIdentity(aaxMtx4f mtx)
Load the Identity matrix into mtx.

Return AAX_TRUE. 

int aaxMatrixTranslate(aaxMtx4f mtx, float dx, float dy, float dz)
Shift the position of the matrix by dx, dy and dz.

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrixRotate(aaxMtx4f mtx, float angle, float x, float y, float z)
Rotate the matrix by angle radians around the axis defined by (x, y, z).

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrixMultiply(aaxMtx4f mtx, aaxMtx4f mtx2)
Multiply both matrices mtx and mtx2 and store the result in mtx

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrixInverse(aaxMtx4f mtx)
Calculate the inverse of the matrix and store the result in mtx.

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrixToMatrix64(aaxMtx4d mtx1, aaxMtx4f mtx2)
Convert the 32-bit floating-point matrix mtx2 to a 64-bit floating-point matrix mtx1.

Return AAX_TRUE if successful, AAX_FALSE otherwise.
