Matrix Position and Orientation

int aaxMatrix64SetDirection(aaxMtx4d mtx, const aaxVec3d pos, const aaxVec3f at)
Construct a matrix based on the pos and at vector parameters.

  • If either the pos or at parameters is NULL or contains NaN's the direction is not changed but the program shall continue without a problem.

Return AAX_TRUE if successful, AAX_FALSE otherwise. 

int aaxMatrix64SetOrientation(aaxMtx4d mtx, const aaxVec3d pos, const aaxVec3f at,
                                             const aaxVec3f up)
Construct a matrix based on the pos, at vector and up vector parameters.

  • If one or more of the pos, at or up parameters are NULL or contains NaN's the orientation is not changed but the program shall continue without a problem.

Return AAX_TRUE if successful, AAX_FALSE otherwise.
