Change Setup parameters

bool aaxMixerSetSetup(aaxConfig config, enum aaxSetupType type, unsigned int setup)

These functions change the respective setup.

  • The config parameter is the handle as returned by aaxDriverOpen.
  • The type parameter is an enumerated type and specifies which setup parameter to change.
  • The setup parameter is type parameter specific (see aaxSetupType for more information).

Return true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed class handle or an incompatible setup value.

 

If hardware limitations prevent a particular request then a closest match will be selected.

Supported values for the type parameter
AAX_FREQUENCY
Request a sample frequency. Between 1000 Hz and 192 kHz.
AAX_TRACKS
Requested this number of tracks. Between 1 and 8.
AAX_FORMAT
Set sample format (see aaxFormat for more information).
Only the System Native formats are supported by the mixer.
AAX_TRACK_SIZE
Request a number of bytes that will be available for the mixer (which alters the refresh rate).
AAX_REFRESH_RATE
Set the refresh rate of the mixer. Between 5 Hz and 200 Hz, defaults to 20 Hz.
The refresh rate influences the latency, the higher the refresh rate the lower the latency.
AAX_MONO_EMITTERS
Reserve this number of mono emitters.
AAX_STEREO_EMITTERS
Reserve this number of stereo emitters.
AAX_AUDIO_FRAMES
Reserve this number of audio-frames.
AAX_UPDATE_RATE
Request this update rate of the mixer. Between 5 Hz and 200 Hz, defaults to 20 Hz.
The update rate defines how often 3D properties (like position and orientation) are updated and 20 Hz should be adequate.
AAX_BITRATE
Set the bitrate (currently MP3 out only)
AAX_TRACK_LAYOUT
The setup parameter sets the track layout for capturing real sensors:
  • AAX_TRACK_ALL: Capture all tracks separately.
  • AAX_TRACK_LEFT: Capture the left track only.
  • AAX_TRACK_RIGHT: Capture the right track only.
  • AAX_TRACK_MIX: Mix all tracks after capturing.
AAX_CAPABILITIES
Set the rendering mode for AAXS encoded instruments. 
bool aaxMixerAddBuffer(aaxConfig config, aaxBuffer buffer)
Adds an audio buffer to the mixer.

  • Only AAXS buffers containing filter and/or effect information are supported.
  • If a new buffer is added all previous defined filters and effects might be erased depending on the buffer specification.
  • If the buffer parameter is NULL or a non existent buffer the program shall continue without a problem.

Return true if successful, false otherwise.

Reasons for failure are an incorrect or already destroyed configuration or buffer handle, adding a buffer which has no AAXS data assigned to it or insufficient memory to add the buffer to the mixer.
