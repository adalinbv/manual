Sensor defines

aaxSensorSetIdentityMatrix(config)
Load the Identity matrix as the sensor position-and-orientation matrix.
aaxSensorCaptureStart(config)
aaxSensorSetState(config, AAX_CAPTURING)
Start capturing audio samples.

  • Samples are captured from a physical device if the config parameter was returned by a call to aaxDriverOpen using AAX_MODE_READ.
  • Samples are captured from the internal mixer in all other cases. 
aaxSensorCaptureStop(config)
aaxSensorSetState(config, AAX_STOPPED)
Stop capturing audio samples. 
aaxSensorGetOffsetSamples(config)
aaxSensorGetOffset(config, AAX_SAMPLES)
Return the offset in samples from the start of the buffer stack. 
aaxSensorGetOffsetFrames(config)
aaxSensorGetOffset(config, AAX_FRAMES)
Return the offset in frames from the start of the buffer stack. 
aaxSensorGetOffsetBytes(config)
aaxSensorGetOffset(config, AAX_BYTES)
Return the offset in bytes from the start of the buffer stack.
