Version information

Version information is available to determine if the loaded library version matches the required version and to detect if certain features are supported In general matching major version numbers makes sure all functions are supported in the expected way. Different minor version numbers could indicated bug fix releases or very minor functionality updates. 
unsigned int aaxGetByType(enum aaxSetupType type)
  • type is an enumerated type aaxSetupType.

Returns the value for the type.

This function replaces aaxGetMajorVersion, aaxGetMinorVersion, aaxGetPatchLevel, aaxMaxFilter and aaxMaxEffect from version 3.0. 

int64_t aaxGetByName(const char *name, enum aaxTypeName type)
  • type is an enumerated type aaxTypeName.

Returns the value for the type by it's name.

This function replaces aaxGetVersionString, aaxGetCopyrightString, aaxGetTypeByName, aaxGetWaveformTypeByName, aaxGetFrequencyFilterTypeByName, aaxGetDistanceModelByName from version 3.0. 

bool aaxPlaySoundLogo(const char *devname)
Plays the official AeonWave sound logo at the device specified by devname. If devname is NULL the default device will be used.
The function will block until playback is finished which takes about 5 seconds.

The function will return false in case of an error and the appropriate error number will be set.
Otherwise true will be returned

Generic

void aaxFree(void *region)
Frees a memory block that was allocated by the library.
Calling this function to free a block of memory is required when the main application is compiled using a different compiler than the AeonWave library. In this case freeing a memory region might (and probably will) result in a segmentation fault. It is therefore recommended to use this function to free all memory that was allocated by AAX for instance after calling aaxBufferGetData.
