Filter Types

Filters and effects are applied in the following order:
  1. velocity effect
  2. pitch effect - fixed pitch
  3. dynamic pitch effect
  4. parent pitch
  5. timed pitch effect
  6. pitch effect - slide
  7. pitch effect - clamping
  8. volume filter - occlusion
  9. bitcrusher filter
  10. ringmodulator effect
  11. frequency filter
  12. bitcrusher filter - add noise
  13. distortion effect
  14. delay effect - phasing, chorus, flanging
  15. delay-line effect
  16. reverb
  17. note velocity
  18. timed gain
  19. note soft, note pressure
  20. parent gain effect
  21. dynamic gain filter
  22. buffer gain
  23. distance filter, directional filter
  24. volume filter
  25. dynamic layer filter
  26. distance attenuation frequency filtering
  27. volume filter - clamping
  28. frame dynamic gain filter
  29. convolution
  30. compressor
  31. equalizer
