Change Setup parameters

int aaxBufferSetup(aaxBuffer buffer, enum aaxSetupType type, unsigned int setup)

These functions change the respective setup.

  • The buffer parameter is the handle as returned by aaxBufferCreate.
  • The type parameter is an enumerated type and specifies which setup parameter to change.
  • The setup parameter is type parameter specific (see aaxSetupType for more information).

Return AAX_TRUE if successful or AAX_FALSE otherwise.

Reasons for failure are an incorrect or already destroyed class handle or an incompatible setup value. 


unsigned int aaxBufferGetSetup(const aaxBuffer buffer, enum aaxSetupType type)

  • The buffer parameter is the handle as returned by aaxBufferCreate.
  • The type parameter specifies which setup parameter to return.

Return the current setup of the class as an enumerated type aaxSetupType or AAX_FALSE if a error occurred.

Reasons for failure are an incorrect or already destroyed class handle or an improper type parameter.

 

Before Setting the Data for the buffer it is required to specify the format and number of samples (or tracks size).

It is possible to specify a different sample frequency and a different data format before getting the data from the buffer. In this case the software automatically convert to the requested data format and will resample to the new sample frequency.

Supported values for the type parameter
AAX_FREQUENCY
Get or Set the sample frequency. Between 1000 and 96000 Hz
AAX_TRACKS
Get or Set the number of tracks that will be available in the buffer. Between 1 and 8.
AAX_FORMAT
Get or Set the sample format of the buffer (see aaxFormat for more information).
AAX_TRACK_SIZE
Get or Set the number of bytes that will be available in the buffer.
AAX_NO_SAMPLES
Get or Set the number of samples that will be available in the buffer.
AAX_LOOP_START
Get or Set the loop start point in samples from the buffer start.
Defaults to 0 samples.
AAX_LOOP_END
Get or Set the loop end point in samples from the buffer start.
Defaults to AAX_NO_SAMPLES.
AAX_BLOCK_ALIGNMENT
Set the number of samples per data block for each channel: 36 for AAX_IMA4_ADPCM.
Defaults to 1.
