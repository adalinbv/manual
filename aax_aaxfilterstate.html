Filter state information

AAX_EQUALIZER
AAX_GRAPHIC_EQUALIZER
AAX_TRUE to enable the filter or AAX_FALSE to disable it.

AAX_FREQUENCY_FILTER

Defines the filter type and filter slope or AAX_FALSE to disable it.

  • For filter sweeping one of the types for AAX_DYNAMIC_GAIN_FILTER as defined below can be added.
    For envelope following (auto-wah) add AAX_ENVELOPE_FOLLOW.
  • The filter type is defined by adding AAX_BUTTERWORTH or AAX_BESSEL to the filter slope.
    e.g.: (AAX_4TH_ORDER|AAX_BESSEL|AAX_ENVELOPE_FOLLOW)
Valid types for the filter slope are:
AAX_6DB_OCT (or AAX_1ST_ORDER)
AAX_12DB_OCT (or AAX_2ND_ORDER)
AAX_24DB_OCT (or AAX_4TH_ORDER)
AAX_36DB_OCT (or AAX_6TH_ORDER)
AAX_48DB_OCT (or AAX_8TH_ORDER)

AAX_DYNAMIC_GAIN_FILTER
AAX_BITCRISHER_FILTER
A waveform type (or AAX_TRUE which equals to AAX_CONSTANT_VALUE) to enable the dynamic filter or AAX_FALSE to disable it.
Randomness is low-frequency noise with a cut-off frequency around the lfo-frequency value.
Timed-transition is a one time action from a start value to an end value which takes (1/lfo-frequency) seconds to complete.
Envelope following calculates the current amplitude and sets the value based on that with an update speed of (1/lfo-frequency) seconds. A low value for lfo-frequency therefore means it takes longer for the envelope following to reach the next value.

Valid waveform types are:
AAX_CONSTANT_VALUE
AAX_TRIANGLE_WAVE
AAX_SINE_WAVE
AAX_SQUARE_WAVE
AAX_SAWTOOTH_WAVE
AAX_RANDOMNESS
AAX_TIMED_TRANSITION
AAX_ENVELOPE_FOLLOW
AAX_INVERSE_TRIANGLE_WAVE
AAX_INVERSE_SINE_WAVE
AAX_INVERSE_SQUARE_WAVE
AAX_INVERSE_SAWTOOTH_WAVE
AAX_INVERSE_TIMED_TRANSITION
AAX_INVERSE_ENVELOPE_FOLLOW

AAX_DISTANCE_FILTER

Possible state parameter values:
AAX_DISTANCE_MODEL_NONE
No distance attenuation at all, attenuation is up to the developer.
The emitter will sound equally loud at any distance.
AAX_EXPONENTIAL_DISTANCE
Natural exponential distance attenuation without distance delay.
AAX_ISO9613_DISTANCE
ISO9613-1 based distance attenuation without distance delay.
AAX_EXPONENTIAL_DISTANCE_DELAY
Natural exponential distance attenuation including distance delay.
AAX_ISO9613_DISTANCE_DELAY
ISO9613-1 based distance attenuation including distance delay.
