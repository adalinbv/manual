Debugging functions

enum aaxErrorType aaxGetErrorNo()
Returns the number of the last error as an enumerated type aaxErrorType and clears the error state to AAX_ERROR_NONE. 
const char* aaxGetErrorString(enum aaxErrorType error_no)
Returns error information about the specified error number as a string constant. 
bool aaxIsValid(const void* handle, enum aaxHandleType type)
Test whether a handle is still valid or not.

  • The handle parameter is either a driver, emitter, buffer or audio-frame handle.
  • The type parameter is an enumerated type aaxHandleType and specifies which handle type to test for.

Return true if the handle is valid or false otherwise.
