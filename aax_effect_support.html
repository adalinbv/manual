bool aaxEffectGetSlot(aaxFilte effect, unsigned slot, int ptype,
                                              float *p1, float *p2, float *p3)
Return all parameters of a defined parameter-slot for an effect.

  • The effect parameter is the handle as returned by aaxEffectCreate or one of the class getter functions.
  • The slot parameter defines which slot number to return.
    Slot numbers start at zero.
  • The ptype parameter specifies the unit of the parameter and is type specific.
    See enum aaxType for more information.

Returns true if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed effect handle or an incorrect slot number. 

float aaxEffectGetParam(aaxEffect effect, int param_no, int ptype)
Get the value of the specified parameter of the effect.

  • The effect parameter is the handle as returned by aaxEffectCreate or one of the class getter functions.
  • The param_no parameter defines which value of the effect parameters to return.
    See enum aaxParameter for more information.
  • The ptype parameter specifies the unit of the parameter and is type specific.
    See enum aaxType for more information.

Return the parameters value if successful or false otherwise.

Reasons for failure are an incorrect or already destroyed effect handle or an incorrect param_no value.
