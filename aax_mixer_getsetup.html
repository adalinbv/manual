Change Setup parameters

unsigned int aaxMixerGetSetup(const aaxConfig config, enum aaxSetupType type)

  • The config parameter is the handle as returned by aaxDriverOpen.
  • The frame parameter is the handle as returned by aaxAudioFrameCreate.
  • The type parameter specifies which setup parameter to return.

Return the current setup of the class as an enumerated type aaxSetupType or AAX_FALSE if a error occurred.

Reasons for failure are an incorrect or already destroyed class handle or an improper type parameter.

Supported values for the type parameter
AAX_FREQUENCY
Get the sample frequency.
AAX_TRACKS
Get the number of tracks.
AAX_FORMAT
Get sample format of the output device (see aaxFormat for more information).
AAX_TRACK_SIZE
Get the number of bytes that are be available in the buffer.
AAX_REFRESH_RATE
Get the refresh rate of the mixer in Hz.
AAX_MONO_EMITTERS
Get the number of mono emitters available.
AAX_STEREO_EMITTERS
Get the number of stereo emitters available.
AAX_AUDIO_FRAMES
Get the number of audio-frames available.
AAX_UPDATE_RATE
Get the update rate of the mixer in Hz.
The update rate defines how often 3D properties (like position and orientation) are updated.
AAX_BITRATE
Get the bit rate (currently MP3 only)
AAX_PEAK_VALUE
Returns the peak value of a specified track after mixing. Between 0 and 8388608.
Select a track by using a logical OR and an aaxTrackType, e.g. (AAX_PEAK_VALUE | AAX_TRACK_RIGHT).
AAX_AVERAGE_VALUE
Returns the RMS average value of a specified track after mixing. Between 0 and 8388608.
Select a track by using a logical OR and an aaxTrackType, e.g. (AAX_AVERAGE_VALUE | AAX_TRACK_LEFT).
AAX_COMPRESSION_VALUE
Returns the compression level of a specified track after mixing. Between 0 and 8388608.
Only useful if the Compressor or a Dynamic Gain filter was applied.
Select a track by using a logical OR and an aaxTrackType, e.g. (AAX_COMPRESSION_VALUE | AAX_TRACK3).
AAX_LATENCY
Get the latency in microseconds.
AAX_FRAME_TIMING
Get the time it took to render the last frame in µs.
AAX_GATE_ENABLED
Returns AAX_FALSE if the noise-gate is not active for a specified track.
Select a track by using a logical OR and an aaxTrackType, e.g. (AAX_GATE_ENABLED | AAX_FRONT_RIGHT).
AAX_SHARED_MODE
Returns AAX_FALSE if the device cannot be shared with other applications.
AAX_TIMER_MODE
Returns AAX_FALSE if the device only supports interrupts for operation.
AAX_BATCHED_MODE
Returns AAX_FALSE if the device can not operate in maximum speed rendering mode.
AAX_TRACKS_MIN
Get the minimum number of tracks supported by this device.
AAX_TRACKS_MAX
Get the maximum number of tracks supported by this device.
AAX_FREQUENCY_MIN
Get the minimum sample frequency supported by this device.
AAX_FREQUENCY_MAX
Get the maximum sample frequency supported by this device.
AAX_SAMPLES_MAX
Get the maximum number samples available in the File.
AAX_PERIODS_MIN
Get the minimum number of buffer periods for this driver.
AAX_PERIODS_MAX
Get the maximum number of buffer periods for this driver.
