enum aaxRenderMode

AAX_MODE_READ
Audio capturing mode
AAX_MODE_WRITE_STEREO
Stereo output mode.
This is the only mode that won't take precedence over the mode specified by one of the configuration files.
AAX_MODE_WRITE_SPATIAL
Spatialized stereo output that extends beyond the location of the speakers.
AAX_MODE_WRITE_SURROUND
Typical surround sound setup.
AAX_MODE_WRITE_HRTF
Spatialized headphone setup with additional source location cues.
