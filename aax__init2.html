enum aaxFilterType aaxMaxFilter(void)
  • returns the maximum number of supported filter types. 
enum aaxEffectTyp aaxMaxEffect(void
  • returns the maximum number of supported effect types. 
bool aaxIsFilterSupported(aaxConfig config, const char *filter)
bool aaxIsEffectSupported(aaxConfig config, const char *effect)
  • The config parameter is the handle as returned by aaxDriverOpen.
  • The config parameter may be set to NULL, in which case the minimum available configuration (AeonWave-Lite) is used.
  • As of AeonWave version 2.6 filters and effects can have a version number appended to the string name.
    The filter or effect name string parameter may include the version number for this test, but it is not necessary.
    If the version number is specified exactly the function returns only true if it matches exactly.
    If the version number is provided only partially then true is returned if the available filter or effect version has the same major version.
    If no version information is provided to test for the function returns true if the filter or effect is available in any version.
