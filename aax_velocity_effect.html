Velocity Effect

The Velocity Effect affects the pitch of the sound source based on the direction vector of the sound source relative to the direction vector of the virtual sensor. This effect is also known as the Doppler effect.

The Sound Velocity parameter defines the Speed of Sound in the scenery surrounding the virtual sensor or the emitter.

The library interpolates between the specified speed of sound at the emitters location and the specified speed of sound at the virtual sensor taking into account both Doppler Factor parameters.

If the ISO9613-1 distance model is applied, and the velocity effect was not yet set, then the corresponding speed of sound will be set automatically. If the sound velocity is set afterwards then updating the ISO9613-1 distance model will not overwrite the client supplied sound velocity.

The Velocity effect is supported by the Scenery and Emitters.
Velocity Effect parameters
The effect uses only one effect slot.
AAX_VELOCITY_EFFECT
3D
p0
AAX_SOUND_VELOCITY
Sound velocity in the same unit as the velocity vector.
Between 0.0 and max. float.
p1
AAX_DOPPLER_FACTOR
A factor to understate or exaggerate the velocity effect.
Between 0.0 and 10.0, defaults to 1.0
p2
AAX_LIGHT_VELOCITY
Light velocity in the same unit as the sound velocity.
Between 0.0 and max. float.
p3
Unused.
Velocity Effect state options
The effect is always enabled.
