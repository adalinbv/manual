Emitter defines

aaxEmitterGetAudioCone(emitter, inner_angle, outer_angle, outer_gain)
Return the inner-angle and outer-angle in radians and the linear outer-gain of the audio cone using the directional-filter.
If either inner_angle, outer_angle or outer_gain is NULL it's value will not be set. 
aaxEmitterGetRolloffFactor(emitter)
Return the distance attenuation roll-off factor using the distance-filter. 
aaxEmitterGetReferenceDistance(emitter)
Return the distance attenuation reference distance using the distance-filter. 
aaxEmitterGetMaxDistance(emitter)
Return the distance attenuation maximum distance using the distance-filter.
