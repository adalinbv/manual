Effect Types

Filters and effects are applied in the following order:
  1. Directional Filter
  2. Velocity Effect
  3. Pitch Effect
  4. Timed Pitch Effect
  5. Dynamic Pitch Effect
  6. Distance Filter
  7. Timed Gain Filter
  8. MIDI Note Soft and Note Pressure
  9. Dynamic Gain Filter
  10. Volume Filter
  11. Dynamic Layer Filter
  12. BitCrusher Filter
  13. RingModulator Effect
  14. Frequency Filter
  15. Bitcrusher Filter - add noise
  16. Distortion Effect
  17. Delay Effect (Phasing/Chorus/Flanging)
  18. Delay-line Effect
  19. Reverb or Convolution
  20. Compressor
  21. (Graphic) Equalizer
  22. Limiter
