Buffer static setup

aaxBuffer aaxBufferCreate(aaxConfig config, unsigned int no_samples,
                            unsigned no_tracks, enum aaxFormat format)
Create a new audio buffer using the supplied parameters.

  • The config parameter is the handle as returned by aaxOpenDevice.
    It will not be possible to use the aaxBufferProcessWaveform function to generate waveforms on-demand if NULL is passed as the config parameter, otherwise buffers will function properly.
  • The no_samples parameter specifies the number of samples the buffer should be able to hold for every track.
  • The no_tracks parameter specifies the number of tracks for this buffer.
  • The format parameter specifies which format or compression type the data will be in.
    The format parameter is an enumerated type aaxFormat.
  • The sample frequency of the buffer defaults to 48kHz and should be set to the correct value using aaxBufferSetSetup after calling this function.

Return the new audio buffer or NULL in case of an error.

Reasons for failure are an incorrect or already destroyed config handle, an unsupported format parameter, the no_samples or no_tracks parameter being zero or insufficient memory to allocate the buffers. 

aaxBuffer aaxBufferReadFromStream(aaxConfig config, const char *url)
Creates a new audio buffer from a file indicated by the url.
  • The config parameter is the handle as returned by aaxOpenDevice.
  • The url could either be a local file or a file on a remote web-server.
  • All recognized streaming formats are supported which includes: WAV, MP3, Vorbis and AAXS.
Return the new audio buffer or NULL in case of an error.

Reasons for failure are an incorrect or already destroyed config handle, an invalid URL or an unsupported audio format. 

bool aaxBufferDestroy(aaxBuffer buffer)
Destroy the audio buffer and all associated memory buffers.

  • The buffer parameter is the handle as returned by aaxBufferCreate.

Return true if successful, false otherwise.

Reasons for failure are an incorrect or already destroyed buffer handle.
