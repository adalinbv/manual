Position and Orientation

Positions may be absolute, or relative to the sensor position. This is done either by calling aaxEmitterSetMode or aaxAudioFrameSetMode.

The orientation is defined with the x-axis positive to the right, the y-axis positive upwards and the z-axis positive backwards.

The sensor matrix should be inversed using aaxMatrix64Inverse first if the matrix was manipulated using the matrix support functions.

AeonWave does not define the units of measurement for distance, velocity or size, the calculations are scale invariant.
Developers have to make sure that all distances and sizes used by AeonWave use the same unit.
