Class AeonWave

Header file: <aax/aeonwave.hpp>
Namespace: aax
Inherits: Sensor

 

The AeonWave class encapsulates the Driver, Mixer, Scenery and Sensor functions.

It allows setting and getting the State, Mode and Setup parameters and Filter and Effects for all four classes by calling the set() function calls or get() function calls with the appropriate enumerated type. It also handles registering and de-registering of Emitters, Audio-Frames and Sensors, retrieving of information about the selected driver and adds buffer management code and support for playing one fire-and-forget background music stream.
